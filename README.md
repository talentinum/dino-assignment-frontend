# Dino Membership 프론트엔드 엔지니어 과제

프론트엔드 엔지니어용 과제입니다.

간단한 Todo 웹앱의 일부가 구현되어 있습니다. 소스코드를 참조하시어 아래 컴포넌트를 완성해주세요.

- `app/pages/Add.tsx`
- `app/pages/Edit.tsx`

## 실행

아래 명령어로 테스트 서버를 실행해볼 수 있습니다.

```bash
npm run run:dev
```
